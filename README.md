## Travaux Pratiques 1 : Comprendre l'asynchrone avec Future en Dart et API REST avec Node.js - Prof : Mehdi Ait Said

## Description

Ce projet est un exercice pratique qui illustre la gestion des produits et des commandes à l'aide d'une API REST développée avec Node.js. Un client Dart est également inclus pour interagir avec cette API de manière asynchrone.

## Structure du Projet

```plaintext
tp-backend/
├── server.js                # Implémentation du serveur en Node.js avec Express
├── data.json                # Fichier JSON pour stocker les produits et les commandes
└── package.json             # Fichier de configuration du projet Node.js

api_client/
├── client.dart              # Client Dart pour consommer l'API
└── pubspec.yaml             # Fichier de configuration du projet Dart, y compris les dépendances
